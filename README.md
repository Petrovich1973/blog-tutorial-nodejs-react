# blog-tutorial-nodejs-react

Creating blog archive on Nodejs, Express, React, Redux, Webpack, ES6/ES7
===================

Usage
-----

Install:

.. code:: bash

  $ npm i -g babel-cli
  $ npm i -g nodemon
  $ npm i -g webpack
  $ npm i

Run Server:

.. code:: bash

  $ npm start

Run Webpack:

.. code:: bash

  $ npm run front
